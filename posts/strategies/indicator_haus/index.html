平均足アップスイング戦略

平均足アップスイング戦略は、平均足を活用して上昇する傾向に追従し、上昇場でのみ収益を実現する短期売買に適した戦略です。

重要事項
  • PROバージョンのユーザーのみ無料で利用できます。
  • 推奨する時間足は30分、1時間、2時間で、現物取引でビットコイン、イーサリアムなど流動性の高いコインに適しています。
  • 最初は少額でご使用いただきありがとうございます。
  • インジケーター の使用によって発生した売買損失については一切責任を負いませんのでご注意ください。

説明

平均足アップスイング戦略は、平均足を活用して上昇する傾向に追従し、上昇場でのみ収益を実現する短期売買に適したトレンド追従戦略で、ロングにのみ使用可能で現物に適しています。
最大の特徴は、Pineスクリプトのストラテジーをソース修正なしですぐに自動販売に使用できるため、Pineスクリプトを知らなくても誰でも簡単に使用できます。

入力項目説明


入力項目
  • Authentification Key :認証キーを入力
  • Long Entry OrderID : 買い注文IDを入力
  • Long Close OrderID : 売り注文IDを入力
  • Short-Term Volatility (初期値:96) : 短期ボラティリティ入力 (※1)
  • Long-Term Volatility (初期値:97) : 長期ボラティリティ入力 (※1)
  • Take Profit (%) (初期値:15) : 利食(%)を入力
  • Trailing Stop (%) (初期値:10) : トレーリングストップを入力
(※1) 値が低いほど買いエントリが積極的

注文メッセージ作成

Binance BTC/USDTを例に説明します。

買い注文メッセージ

1.注文メッセージ作成画面で、下記のように買い注文に必要な内容を入力し、メッセージ登録ボタンをクリックします。

※注文IDはメッセージを登録すると自動生成されます。 ①と②はチャート設定で使用します。

例) 買い注文メッセージの内容
  • 取引所 : Binance
  • アカウント : デフォルト
  • 通貨ペア : BTC/USDT
  • 注文方法 : 買い
  • 注文タイプ : 成行
  • 注文数量 : 100(個 USDT선택)
  • メモ : [平均足アップスイング][Binance現物][BTCUSDT]買い注文

売り注文メッセージ

2.注文メッセージ作成画面で、下記のように売り注文に必要な内容を入力し、メッセージ登録ボタンをクリックします。

※注文IDはメッセージを登録すると自動生成されます。 ③と④はチャート設定で使用します。

例) 売り注文メッセージの内容
  • 取引所 : Binance
  • アカウント : デフォルト
  • 通貨ペア : BTC/USDT
  • 注文方法 : 売り
  • 注文タイプ : 成行
  • 注文数量 : 保有残高比率100(％)
  • メモ : [平均足アップスイング][Binance現物][BTCUSDT]売り注文

チャート設定

注意事項

買いアラットの設定時に買いシグナルが発生して買い注文が実行される可能性があります。
そのため、必ず下記のように該当取引所のAPI接続を中止にしてからアラットを設定してください。

【APIキー中止状態】


アラート追加を完了した後、必ずAPIを接続中にしてください。

【APIキー接続中状態】

3.まずお使いの取引所と１時間足を選択し、平均足ップスイング指標を呼び出します。
(※ 例はBinance現物１時間足チャートです。) ※インジケーター 検索ウィンドウに『[TVExtBot]Heikinashi Upswing Strategy』と入力


4.チャートにインジケータを追加すると、自動的に入力画面が開き、下のように上記の注文メッセージ作成で登録された注文メッセージの注文IDとメモを入力します。 ※注文IDとメモは注文メッセージ作成画面にあります をクリックするとコピーされます。

注文IDとメモを入力する
  • Authentification Key (認証キー) : 認証キーを入力
  • Long Entry OrderID (買い注文ID) : ①を入力 Memo : ②を入力
  • Long Close OrderID (売り注文ID) : ③を入力 Memo : ④を入力
注文IDとメモ入力後画面
重要事項

自動販売中に上記の入力項目を変更する場合、変更した内容を反映するには、必ず既存のアラートを削除して新しいアラートを追加する必要があります。

アラート設定

アラート設定入力内容
  • ⑤ 条件 : 『[TVExtBot]HAUS』 を選択
  • ⑥ Webhook URL : 『https://tvextbot-trading.web.app/api/webhook』 ウェブフックURLを入力 (※1)
  • ⑦ アラート名 : 『平均足アップスイング戦略(Binance現物BTC)』 を入力 (※未入力でも問題ない)
  • ⑧ メッセージ : 『{{strategy.order.alert_message}}』 を入力 (※2)
(※1) 必ず本人のメッセージ作成画面にあるウェブフックURLを入力してください。
(※2) 以下の内容をコピーして、アラートにメッセージに貼り付けてください。 
{{strategy.order.alert_message}}
