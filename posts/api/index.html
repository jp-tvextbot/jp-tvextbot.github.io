取引所APIの発行方法

各取引所のAPI発行方法について説明します。

現在TVExtBot内で利用可能な取引所
  • Liquid
  • Zaif
  • Bybit 現物、先物(USD、USDT)
  • Binance 現物、USDT先物、Coin先物
  • FTX 現物、先物
  • BitMEX
  • Bitget USDT先物、Coin先物
  • Phemex 現物、先物
  • MEXC 先物
  • Bybit テストネット
  • BitMEX テストネット
  • Binance 先物 テストネット
  • Phemex テストネット
  • Bitget 先物 テストネット
APIキー発行時に固定IP登録が必要な場合、本人のAPIキー登録画面にあるIPアドレスを必ず登録してください。 （有料バージョンのみ）
ウェブフックURLとIPアドレスはユーザーごとに異なります。 ぜひご自身の拡張プログラム設定画面にあるウェブフックURLとIPアドレスをご使用ください。
Bybit APIキーの発行方法

   APIキー管理画面で「Create New Key」をクリックすると、以下の画面が表示されます。

   上記のように入力して「Submit」をクリックし、生成されたAPIキーと秘密キーをコピーしてAPI登録画面に登録してください。
   秘密キーは画面を閉じると確認できないので必ずメモしておいてください。
   出金は絶対にチェックしないでください!

Binance APIキーの発行方法

   APIキー管理画面でラベル名を記入した後、「Create API」をクリックすると以下の画面が表示されます。

   「Edit」をクリックして上記のように入力した後、生成されたAPIキーと秘密キーをコピーしてAPI登録画面に登録してください。

   出金は絶対にチェックしないでください!
